0. UPDATES AND PATCHES
Play Epik Videogames receives updates to fix problems within the game or security patches on its servers, and that's why we would like to better explain how to update our Videogames to the latest version in the right way, below we will point out the steps.

1. Visit our itch or gamejolt site and download the right package for your computer, to see if your machine has a 64-bit or 32-bit processor:
WINDOWS 11/10 : Device Settings > System > Go all the way down and find the "system information" item > find the "System Type" item and that's where you can find out if your computer has a 64-bit or 32-bit processor.

2. If you have another version of the product installed, uninstall the previous version first and then start the installer of the new version.

3. If you have not installed any version prior you can proceed with opening the end named ("SetupX" or "GameNameSetupX") + device bit, following the information on the screen you will be able to install the videoGame.
