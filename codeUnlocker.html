A SHORT DESCRIPTION

Code Unlocker is a BrainStorming game, where the goal is to find the right combination of numbers by observing very carefully the question that the level requires. Each level is different and more difficult than the previous one! until you get to level 110, the last level of Code Unlocker. If you have finished all 110 levels and want to play again, rest assured that you will earn UNLIMITED PLAY, a reward with which you can choose the level and play it without limits!
