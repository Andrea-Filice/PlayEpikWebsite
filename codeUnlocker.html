CODE UNLOCKER.

A game where the password  is important.










A SHORT DESCRIPTION

CODE UNLOCKER is a BRAINSTORMING GAME, where the goal is to find the right combination of numbers by observing very carefully the question that the level requires. Each level is different and more difficult than the previous one! Until you get to level max. If you have finished all 110 levels and want to play again, rest assured that you will earn UNLIMITED PLAY, a reward with which you can choose the level and play it without limits!
